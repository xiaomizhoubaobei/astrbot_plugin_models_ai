# 证件照自动生成功能文档

## 功能概述
该功能根据用户上传的人像照片，自动生成符合各类标准的证件照，包括尺寸裁剪、背景替换、美化等功能。

## 实现方式
- 集成人像检测API
- 结合背景替换和图像美化功能
- 利用现有图像处理管道

## 命令接口
```
/ai-gitee id-photo <人像照片> [证件类型]
```

## 证件类型
- passport (护照照片)
- id-card (身份证照片)
- visa (签证照片)
- driving-license (驾驶证)
- resume (简历照片)
- school (校园证件)

## 输入参数
- 人像照片
- 证件类型（可选，默认passport）
- 背景色（蓝底、红底、白底）
- 输出质量（标准、高清、印刷）

## 处理功能
- 智能人脸定位
- 标准尺寸裁剪
- 背景颜色替换
- 面部轻微美化
- 光线均衡调整

## 输出规格
- 护照照片: 35x49mm, 300dpi
- 身份证照片: 26x32mm, 350dpi
- 简历照片: 35x45mm, 300dpi
- 支持自定义尺寸

## 智能美化
- 去除红眼和瑕疵
- 适度美白和磨皮
- 优化光线阴影
- 保持真实面貌

## 输出结果
- 符合标准的证件照
- 可直接用于官方申请
- 包含规格说明信息

## 错误处理
- 人脸角度不正确时提示调整
- 光线条件不佳时建议重拍
- 头部比例不符时提供指导

## 适用场景
- 各类证件申请
- 简历照片制作
- 在线认证头像
- 各种登记用途
