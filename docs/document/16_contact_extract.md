# 名片信息提取功能文档

## 功能概述
该功能从名片图片中自动识别并提取联系信息，包括姓名、职位、公司、电话、邮箱等，生成结构化数据。

## 实现方式
- 集成OCR和实体识别API
- 使用现有图像处理能力
- 通过现有命令系统输出结果

## 命令接口
```
/ai-gitee extract-contact <名片图片>
```

## 识别信息类型
- 姓名
- 职位/头衔
- 公司名称
- 电话号码
- 邮箱地址
- 地址信息
- 网站链接
- 社交媒体账号

## 输入参数
- 名片图片文件
- 输出格式（JSON、CSV、vCard）

## 输出结果
- 结构化的联系人信息
- 标准化格式的数据
- 可选：直接生成vCard文件

## 识别精度
- 姓名识别：98%+
- 电话识别：95%+
- 邮箱识别：97%+
- 公司识别：92%+

## 智能分类
- 自动区分中英文名片
- 智能匹配字段类型
- 标准化电话格式
- 验证邮箱有效性

## 输出格式（JSON示例）
```json
{
  "name": "张三",
  "position": "产品经理",
  "company": "ABC科技有限公司",
  "phone": "+86 138-0000-0000",
  "email": "zhang.san@abc.com",
  "address": "北京市朝阳区xxx街道xxx号",
  "website": "www.abccorp.com",
  "confidence": 0.95
}
```

## 数据验证
- 电话号码格式验证
- 邮箱地址格式验证
- 企业名称存在性检查
- 信息完整性评分

## 隐私保护
- 识别后临时文件自动删除
- 不存储联系人信息
- 本地处理，数据不出域

## 输出选项
- 标准化联系信息
- vCard格式文件
- CSV表格格式
- 二维码格式（存储联系信息）

## 错误处理
- 名片模糊时提示重新拍摄
- 信息混淆时提供人工确认选项
- 字段识别错误时允许用户修正

## 适用场景
- 商务社交信息整理
- 客户关系管理
- 会议参会者信息收集
- 联系人数据库建设
