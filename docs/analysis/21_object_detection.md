# 物体识别功能文档

## 功能概述
该功能识别图片中包含的具体物体类别及其位置，提供详细的物体清单和置信度信息。

## 实现方式
- 集成物体检测API
- 使用现有图像处理能力
- 通过现有命令系统输出结果

## 命令接口
```
/ai-gitee detect-objects <图片> [选项]
```

## 支持的物体类别
- 人（人、人脸、身体部位）
- 动物（狗、猫、鸟、鱼等）
- 交通工具（汽车、飞机、船、自行车等）
- 电子设备（手机、电脑、电视等）
- 家具（桌子、椅子、床等）
- 食物（水果、蔬菜、菜肴等）
- 自然景观（山、河、树、花等）
- 建筑结构（房屋、桥梁、塔等）

## 输入参数
- 图片文件
- 检测阈值（50%-95%，默认80%）
- 返回数量（1-50个，默认10个）
- 是否包含位置信息（是/否）

## 输出结果
- 检测到的物体列表
- 每个物体的置信度
- 物体在图片中的位置坐标（可选）

## 识别精度
- 常见物体：95%+
- 特殊物体：85%+
- 小物体：80%+
- 遮挡物体：75%+

## 输出格式
```
检测到的物体:
1. 人脸 - 置信度: 98% - 位置: (x1,y1,x2,y2)
2. 手机 - 置信度: 92% - 位置: (x1,y1,x2,y2)
3. 桌子 - 置信度: 89% - 位置: (x1,y1,x2,y2)
```

## 技术特点
- 实时物体检测
- 多尺度检测
- 非极大值抑制
- 边界框精确定位

## 检测优化
- 复杂场景优化
- 小物体增强检测
- 重叠物体分离
- 光线条件适应

## 分类统计
- 物体类型分布
- 数量统计
- 主要物体识别
- 次要物体识别

## 智能分析
- 物体关系推断
- 场景理解
- 活动类型推断
- 环境分析

## 输出选项
- 物体清单列表
- 带坐标的结构化数据
- JSON格式结果
- 统计汇总报告

## 错误处理
- 图片中物体过多时限制返回数量
- 物体难以识别时提供置信度阈值调整建议
- 识别错误时允许用户反馈修正

## 适用场景
- 图片内容审核
- 智能相册分类
- 安全监控分析
- 电商商品识别
