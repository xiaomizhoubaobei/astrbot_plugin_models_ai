# 水印添加/去除功能文档

## 功能概述
该功能支持为图片添加自定义水印，以及智能去除现有水印，保护图片版权或清除不需要的标识。

## 实现方式
- 集成水印生成和去除API
- 利用现有图像编辑功能
- 通过现有客户端管理器调度

## 命令接口
### 添加水印
```
/ai-gitee add-watermark <图片> [水印文本或图片] [位置]
```

### 去除水印
```
/ai-gitee remove-watermark <带水印图片>
```

## 水印类型
- text (文字水印)
- image (图片水印)
- logo (自定义logo)

## 位置选项
- center (中央)
- corner (角落)
- tile (平铺)
- custom (自定义坐标)

## 添加水印参数
- 水印内容（文字或图片路径）
- 位置（默认center）
- 透明度（1-10，默认5）
- 字体大小/图片比例（默认中等）

## 去除水印参数
- 智能识别（自动检测水印区域）
- 手动选择（指定水印区域）
- 修复质量（低、中、高）

## 输出结果
- 添加水印后的图片
- 或去除水印后的清洁图片

## 安全提示
- 仅处理用户上传的图片
- 不可用于侵犯他人版权的去除行为
- 遵守相关法律法规

## 适用场景
- 个人作品版权保护
- 商业图片品牌标识
- 防止未经授权使用
- 清理非原创图片水印
