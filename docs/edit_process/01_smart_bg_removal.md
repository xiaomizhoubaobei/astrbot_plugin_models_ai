# 智能背景移除功能文档

## 功能概述
该功能允许用户上传一张图片，系统将自动识别前景主体并移除背景，生成带有透明背景的PNG图片。

## 实现方式
- 使用现有API接口的图像分割能力
- 调用图像编辑功能，结合masking技术
- 利用现有`edit_image`方法扩展参数

## 命令接口
```
/ai-gitee remove-bg <图片链接或上传图片>
```

## 输入参数
- 图片文件（支持JPG、PNG、WEBP格式）
- 可选参数：精确度等级（低、中、高）

## 输出结果
- PNG格式图片（透明背景）
- 处理后的图片URL或文件

## 错误处理
- 图片无法识别时提示重新上传
- API调用失败时显示重试建议
- 文件过大时提示压缩后重试

## 适用场景
- 制作头像
- 产品抠图
- 合成背景图片
