# 老照片修复功能文档

## 功能概述
该功能专门针对年代久远、有划痕、褪色、模糊的老照片进行AI修复，提升整体品质。

## 实现方式
- 集成专门的老照片修复模型
- 使用现有图片处理管道
- 组合多种图像修复技术

## 命令接口
```
/ai-gitee restore-photo <老照片> [修复等级]
```

## 修复等级
- light (轻度修复，适合轻微老化)
- medium (中度修复，平衡效果与原貌)
- heavy (重度修复，全面修复但可能改变原貌)

## 输入参数
- 图片文件（老照片）
- 修复等级（可选，默认为medium）
- 特殊处理（勾选表示优先修复人脸）

## 修复功能
- 去除划痕和污渍
- 修复撕裂边缘
- 还原褪色色彩
- 增强模糊细节
- 去除灰尘颗粒

## 处理特色
- 保持历史感
- 优先保护人脸区域
- 智能判断修复程度
- 避免过度处理

## 输出结果
- 修复后的高清图片
- 可选：修复前后对比图

## 错误处理
- 照片损坏过于严重时提示专业修复
- 修复效果不佳时提供其他参数选项
- 文件格式不支持时提示转换格式

## 适用场景
- 家庭老照片修复
- 历史资料整理
- 珍贵影像保存
- 个人收藏保护
