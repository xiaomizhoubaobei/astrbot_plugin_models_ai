# 图片智能裁剪功能文档

## 功能概述
该功能可以根据用户需求智能裁剪图片，自动识别人脸或主要内容区域进行精准裁剪。

## 实现方式
- 集成现有图像处理功能
- 使用人脸识别API检测人脸位置
- 应用主体检测算法识别主要内容

## 命令接口
```
/ai-gitee crop <图片链接或上传图片> [比例] [裁剪模式]
```

## 支持的比例
- 1:1 (正方形)
- 4:3 (横向矩形)
- 3:4 (纵向矩形)
- 16:9 (宽屏)
- 9:16 (竖屏)

## 裁剪模式
- face (优先识别人脸区域)
- focus (优先识别内容焦点)
- center (中心区域裁剪)

## 输入参数
- 图片文件
- 裁剪比例
- 裁剪模式（默认为face）

## 输出结果
- 按指定比例裁剪的图片
- 处理后图片的下载链接

## 错误处理
- 无人脸检测时使用默认裁剪
- 比例不合适时提示调整
- 文件格式不支持时提示支持格式

## 适用场景
- 制作社交媒体头像
- 标准尺寸图片裁剪
- 证件照处理
