# 智能抠图功能文档

## 功能概述
该功能使用AI技术精确分离图片前景与背景，生成高质量透明背景图片，适用于复杂场景。

## 实现方式
- 集成图像分割API
- 使用现有图片编辑管道
- 通过现有客户端扩展功能

## 命令接口
```
/ai-gitee抠图 <图片> [抠图精度]
```

## 抠图精度
- fast (快速模式，适合简单背景)
- accurate (精确模式，适合复杂边缘)
- ultra (超高精度，适合精细毛发等)

## 输入参数
- 图片文件
- 抠图精度（可选，默认accurate）
- 边缘羽化值（0-20px，默认5px）
- 背景处理（透明、白色、指定颜色）

## 技术特点
- 语义分割算法
- 边缘细化技术
- 毛发级精度处理
- 背景虚化效果

## 输出结果
- PNG格式透明背景图片
- 可选：抠图蒙版图层
- 精度报告（边缘准确率等）

## 智能优化
- 自动识别人像边缘
- 优化毛发和半透明物体
- 智能处理复杂背景
- 保持图片细节完整性

## 错误处理
- 图片质量过低时提示提高质量
- 主体与背景区分度低时提供预处理建议
- 抠图失败时提供重试选项

## 适用场景
- 电商产品抠图
- 证件照制作
- 设计素材准备
- 创意合成背景
