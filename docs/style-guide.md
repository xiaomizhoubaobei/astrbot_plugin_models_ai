## 概述

风格转换功能使用 Gitee AI 的图像生成模型，通过预设的风格提示词将图片转换为特定风格。支持 84 种不同风格，涵盖手办化、Q版化、Cosplay、场景化、物品化、角色设计、多图生成、特殊效果等多个类别。

## 功能特点

- ✅ **84 种风格**：涵盖手办、Q版、Cosplay、场景、物品、角色设计等多个类别
- ✅ **文生图和图生图**：支持两种模式，灵活使用
- ✅ **自定义描述**：可附加自定义描述，个性化风格
- ✅ **比例支持**：支持 7 种常用比例
- ✅ **智能处理**：AI 自动理解风格要求，智能应用风格效果
- ✅ **高质量输出**：生成高质量的风格转换结果

## 基本用法

### 命令格式

```
/ai-gitee style <风格名称> [自定义描述] [比例]
```

### 使用模式

#### 模式 1：文生图（不发送图片）

根据风格提示词生成新图片。

**示例**：
```
/ai-gitee style 手办化
```

#### 模式 2：图生图（发送图片）

将发送的图片转换为指定风格。

**示例**：
```
[发送一张照片]
/ai-gitee style 手办化
```

### 使用步骤

**文生图**：
1. 输入风格转换命令
2. 等待 AI 生成图片

**图生图**：
1. 发送要转换的图片
2. 输入风格转换命令
3. 等待 AI 转换图片

## 参数说明

### 风格名称（必需）

指定要转换的风格名称，必须是 84 种可用风格之一。

**查看所有风格**：
```
/ai-gitee style
```

### 自定义描述（可选）

附加的自定义描述，可以个性化风格效果。

**示例**：
- `一个可爱的女孩`
- `猫娘，粉色头发，蓝色眼睛`
- `赛博朋克风格的城市`

### 比例（可选）

指定图片的长宽比，决定生成图片的形状。

**支持比例**：
- 1:1, 4:3, 3:4, 3:2, 2:3, 16:9, 9:16

**默认值**：1:1（1024x1024）

## 常见问题

### Q: 如何查看所有可用风格？

**A**: 使用以下命令查看所有可用风格：
```
/ai-gitee style
```

会列出所有 84 种风格名称。

---

### Q: 文生图和图生图有什么区别？

**A**:
- **文生图**：不发送图片，根据风格提示词生成新图片
- **图生图**：发送图片，将图片转换为指定风格

---

### Q: 可以同时使用多种风格吗？

**A**: 不可以。每次命令只能使用一种风格。如需多种风格，请分别调用命令。

---

### Q: 风格转换需要多长时间？

**A**: 转换时间取决于图片大小和复杂度：
- 小尺寸：通常 30-60 秒
- 中等尺寸：通常 60-120 秒
- 大尺寸：通常 120-180 秒

---

### Q: 如何获得更好的风格转换效果？

**A**: 可以通过以下方式：
1. 使用高质量的原始图片（图生图）
2. 添加详细的自定义描述
3. 尝试不同的风格
4. 优化提示词

---

### Q: 风格转换会改变图片内容吗？

**A**:
- **文生图**：根据风格提示词生成新图片，内容可能不同
- **图生图**：保持图片主要内容，只改变风格

---

### Q: 可以自定义风格提示词吗？

**A**: 不可以。风格提示词是预设的，存储在 `style_prompts.json` 文件中。但可以添加自定义描述。

---

### Q: 为什么某些风格效果不好？

**A**: 可能的原因：
1. 风格提示词不够精确
2. 原始图片质量不佳（图生图）
3. 模型理解偏差
4. 提示词描述不够具体

**解决方法**：
- 尝试其他风格
- 优化提示词
- 使用高质量的原始图片

---

### Q: 可以保存风格转换结果吗？

**A**: 可以。生成的图片会保存到插件目录，您可以：
- 直接查看图片文件
- 使用其他工具保存图片
- 插件会定期清理旧图片

---

### Q: 风格转换支持哪些图片格式？

**A**: 支持大多数常见图片格式：
- ✅ PNG
- ✅ JPG/JPEG
- ✅ WEBP
- ❌ 不支持 GIF（动画）

---

### Q: 如何选择合适的风格？

**A**: 根据需求选择：
- 🎨 动漫角色 → 手办化、Q版化
- 👤 真人照片 → Cosplay、真人化
- 🌄 风景照片 → 场景化系列
- 🎁 周边产品 → 物品化系列
- 🎭 角色创作 → 角色设计系列

### Q: 风格转换失败怎么办？

## 高级用法

### 组合多种风格

逐步转换，探索不同风格：

```
# 步骤 1：生成基础图片
/ai-gitee generate 一个可爱的女孩

# 步骤 2：转换为手办风格
[发送上一步生成的图片]
/ai-gitee style 手办化

# 步骤 3：转换为 Q 版风格
[发送上一步生成的图片]
/ai-gitee style Q版化

# 步骤 4：转换为 Cosplay 风格
[发送上一步生成的图片]
/ai-gitee style cos化
```

---

### 创建系列作品

使用同一主题创建系列作品：

```
/ai-gitee style 手办化 一个可爱的女孩
/ai-gitee style Q版化 一个可爱的女孩
/ai-gitee style cos化 一个可爱的女孩
/ai-gitee style 真人化 一个可爱的女孩
```

---

### 多图生成

生成多张不同内容的图片：

```
/ai-gitee style 表情九宫格
```

## 最佳实践

### 1. 选择合适的风格

根据需求选择合适的风格：
- 🎨 动漫角色 → 手办化、Q版化
- 👤 真人照片 → Cosplay、真人化
- 🌄 风景照片 → 场景化系列
- 🎁 周边产品 → 物品化系列
- 🎭 角色创作 → 角色设计系列

### 2. 使用高质量原始图片

- 📷 确保图片清晰、光线良好
- 📷 人物特征清晰
- 📷 避免模糊或过暗的图片

### 3. 添加自定义描述

- 📝 使用具体、清晰的描述
- 📝 描述想要的风格效果
- 📝 可以组合多个元素

### 4. 指定合适的比例

- 📱 手机壁纸：9:16
- 🖥️ 电脑壁纸：16:9
- 👤 社交媒体头像：1:1
- 🌄 风景摄影：3:2
- 👤 人像摄影：2:3

### 5. 逐步优化

- 🔄 不要期望一次就完美
- 🔄 可以多次尝试不同的风格
- 🔄 根据效果调整提示词和风格
