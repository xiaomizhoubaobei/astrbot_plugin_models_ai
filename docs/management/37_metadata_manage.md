# 元数据查看/编辑功能文档

## 功能概述
该功能查看和编辑图片的EXIF、IPTC、XMP等元数据信息，包括拍摄参数、地理位置、版权信息等。

## 实现方式
- 集成元数据处理库
- 使用现有图片管理功能
- 通过EXIF/IPTC/XMP标准协议

## 命令接口
```
/ai-gitee metadata <图片> [操作类型] [参数]
```

## 操作类型
- view (查看元数据)
- edit (编辑元数据)
- remove (删除元数据)
- copy (复制元数据)

## 查看选项
- exif (EXIF信息)
- iptc (IPTC信息)
- xmp (XMP信息)
- all (所有信息)
- gps (GPS信息)
- camera (相机信息)
- copyright (版权信息)

## 输入参数
- 图片文件
- 操作类型
- 指定字段（查看/编辑模式）
- 新值（编辑模式）
- 保留原始（是/否）

## 输出结果
- 完整元数据列表
- 结构化数据显示
- 可读性优化格式
- 修改记录信息

## 支持的EXIF字段
- 相机型号
- 镜头型号
- 光圈值
- 快门速度
- ISO值
- 拍摄日期时间
- GPS坐标
- 焦距
- 闪光灯信息
- 测光模式

## 支持的IPTC字段
- 标题

- 关键词
- 版权声明
- 创建者
- 城市
- 国家
- 类别
- 特殊指令

## 支持的XMP字段
- 标题

- 关键词
